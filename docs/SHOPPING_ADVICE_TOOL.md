# 选购建议工具

## 交付信息

- 工具名：`shopping_advice`
- 工具作用：根据商品品类、预算、用途、偏好和限制条件，提供中立选购建议。
- 支持参数：`product`、`budget_min`、`budget_max`、`usage`、`preferences`、`constraints`。
- 触发场景：用户询问某类商品怎么选、应关注什么参数、预算如何分配或有哪些避坑点。
- 返回类型：微信文本回复，包含预算建议、核心指标、使用场景取舍、常见误区和下单检查清单。
- 异常处理：商品品类缺失、预算格式错误、最低预算大于最高预算、预算小于零。
- 测试用例：工具参数定义、耳机品类建议、未知品类通用建议、场景和限制条件处理、非法预算以及回复中不出现商品链接。

## 能力边界

- 不调用京东、淘宝或其他电商平台 API。
- 不搜索、抓取、推荐或返回任何具体商品链接。
- 不提供实时价格、库存、销量、优惠券、佣金、下单或支付能力。
- 建议用于帮助用户明确选择标准，最终规格、售后政策和结算价格由用户在正规平台核对。
- Agent 可以在同一轮组合调用 `map_search` 和 `weather`，先了解出行、地点或天气场景，再调用本工具给出装备选购建议。

## 参数说明

| 参数 | 必填 | 说明 | 示例 |
|---|---|---|---|
| `product` | 是 | 商品品类或装备类型 | `蓝牙耳机` |
| `budget_min` | 否 | 最低预算，单位元 | `100` |
| `budget_max` | 否 | 最高预算，单位元 | `300` |
| `usage` | 否 | 主要使用场景 | `地铁通勤和日常会议` |
| `preferences` | 否 | 优先满足的功能或体验 | `佩戴舒适、降噪、长续航` |
| `constraints` | 否 | 不能接受的条件、兼容性或尺寸限制 | `不要入耳式，只连接安卓手机` |

## 示例用户输入

```text
预算100到300元，通勤用的蓝牙耳机应该怎么选？我比较在意降噪和续航，不要入耳式。
```

```text
周末准备去露营，第一次买帐篷需要关注哪些参数？
```

## 示例输出

```text
蓝牙耳机选购建议

预算：
- 预算范围：¥100 - ¥300；建议保留部分预算作为配件或价格波动余量

优先关注：
- 佩戴舒适度与稳固性
- 主动降噪和通透模式
- 实际续航与充电方式
- 通话麦克风表现
- 多设备切换与系统兼容性

常见误区：
- 不要只看宣传的降噪深度或编码格式
- 确认高级功能是否限制特定手机品牌
```
